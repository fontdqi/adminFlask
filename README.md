# Flask框架入门
## 简介：
Flask是一个非常小的PythonWeb框架，被称为微型框架；只提供了一个稳健的核心，其他功能全部是通过扩展实现的；意思就是我们可以根据项目的需要量身定制，也意味着我们需要学习各种扩展库的使用。
## 特点：
Django虽然功能比较多，但是相对笨重，开发时间成本远大于Flask。

## 前期准备工作
### 一、安装虚拟环境：
